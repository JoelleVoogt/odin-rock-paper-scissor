# Rock Paper Scissors

A browser-based console game built with vanilla JavaScript. Part of [The Odin Project](https://www.theodinproject.com/lessons/foundations-rock-paper-scissors) Foundations curriculum.

## About

This is a 5-round Rock Paper Scissors game played entirely via the browser console. The player competes against the computer, which picks its choice randomly each round. Scores are tracked throughout and a winner is declared at the end.

## How to play

1. Open `index.html` in your browser
2. Open the browser console (F12 or right-click > Inspect > Console)
3. The game starts automatically — follow the prompts to enter your choice each round
4. After 5 rounds, the final scores and overall winner are shown

## Features

- Random computer choice each round
- Case-insensitive input (`rock`, `ROCK`, `RocK` all work)
- Round-by-round result logging
- Score tracking for both player and computer
- Overall winner announced after 5 rounds
